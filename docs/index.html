Programming Languages

Python

Python is a robust language for data science, known for its simplicity and readability. It lets you lets you work quickly and integrate systems more effectively.I use it for developing AI tools and machine learning models.

Power BI

Power BI is a powerful business intelligence tool used to visualize and convey insights. I leverage this tool to create interactive dashboards for communicating business objectives.

SQL

SQL is a language for managing databases and creating relational tables, providing data for analysis. It is commonly accessed through tools like SQL Server Management Studio (SSMS), and alternatives include MySQL and NoSQL databases.

Tableau

Tableau is a business intelligence tool, similar to Power BI, that provides Tableau Desktop for data access, visualization, and analysis. It offers an intuitive interface for uncovering insights, facilitating fast decision-making with offline capabilities, and ensures secure self-service with trusted data.
